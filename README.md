##### 安装
    安装 `npm install hexo -g`
    升级 `npm update hexo -g`
    初始化 `hexo init`

##### 命令
    新建文章 `hexo n "我的博客" == hexo new "我的博客"`
    发布草稿 `hexo p == hexo publish`
    生成静态文件 `hexo g == hexo generate`
    启动服务预览 `hexo s == hexo server`
    部署 `hexo d == hexo deploy`
    新建页面 `hexo new page "pageName"`
    清除缓存 `hexo clean`
